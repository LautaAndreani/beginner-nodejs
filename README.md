# HTTPS

### Status code

[HTTPCats](https://http.cat/)

1. **200** = All its ok
2. **300** = Redirects to anothers urls
3. **400** = Client side errors - 401 Unauthorized
4. **500** = Backend side errors - 502 Bad gateway
